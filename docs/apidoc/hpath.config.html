hpath.config moduleÔÉÅ

Module defining the configuration settings for the histopathology simulation model.

The configuration settings are defined using Pydantic, and are fully convertable to JSON. To actually run a simulation, the configuration settings (as a hpath.config.Config object) are converted to a hpath.model.Model object, which contain the actual Python objects used for Resource tracking, etc.
