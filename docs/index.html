Cambridge University Hospitals NHS Trust: Dashboards Server

This project implements digital tools for Cambridge University Hospitals NHS Trust. Two applications have been developed so far: a sensor dashboard and a simulation dashboard modelling the Histopathology department at Addenbrooke’s Hospital, Cambridge, UK.

Authors:
